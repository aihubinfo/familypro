كثير من المستخدمين يخلطون بين "اشتراك X" و"اشتراك Grok".  
النتيجة غالبًا: شراء الخطة الخطأ، وسوء تقدير حدود الاستخدام، وحساب تكلفة غير دقيق.

> تاريخ البيانات: 2026-04-10  
> ملاحظة: الأسعار والمزايا تختلف حسب الدولة والوقت والسياسة. المرجع النهائي دائمًا صفحة الدفع الرسمية.

## 1) الفرق الأساسي: منتجان منفصلان

| البند | اشتراك X | اشتراك Grok |
| --- | --- | --- |
| الجهة البائعة | X (x.com) | xAI (grok.com) |
| القيمة الأساسية | مزايا المنصة داخل X | قدرات Grok نفسها (محادثة/توليد/Imagine) |
| العلاقة | قد تتضمن بعض خطط X مزايا Grok | لا تمنحك تلقائيًا مزايا X الكاملة |

الخلاصة: **ليسا نفس الاشتراك ولا نفس الفاتورة**.  
يمكن الجمع بينهما، أو الاكتفاء بأحدهما فقط، حسب نقطة استخدامك الأساسية.

## 2) أين يقع Grok داخل خطط X؟

- قيمة X الأساسية تبقى مزايا المنصة
- Grok غالبًا "ميزة إضافية" ضمن بعض الخطط
- الخطط الأعلى عادة تعطي سقفًا أعلى/أولوية أفضل

تظهر في مركز مساعدة X جداول أسعار مرجعية (خصوصًا للويب في الولايات المتحدة)، لكنها تتغير بمرور الوقت وحسب المنطقة.  
تعامل معها كمرجع مقارنة، وليس كسعر نهائي ثابت.

## 3) كيف نقارن Grok داخل X مع SuperGrok؟

### 3.1 فهم سريع للطبقات

| خطة X | وضع Grok (بشكل عام) | مناسب لمن؟ |
| --- | --- | --- |
| Basic | يختلف حسب المنطقة/الوقت | استخدام خفيف لـX |
| Premium | وصول محسّن إلى Grok في العادة | مستخدم X يومي مع استخدام AI متوسط |
| Premium+ | سقف أعلى وأولوية أفضل غالبًا | مستخدم X كثيف + يحتاج Grok أقوى |

### 3.2 مقارنة بالقدرات: Grok داخل X مقابل SuperGrok

| البعد | Grok داخل X (Premium/Premium+) | SuperGrok عبر grok.com |
| --- | --- | --- |
| نقطة الدخول | داخل تطبيق/موقع X | grok.com أو تطبيق Grok |
| السقوف وقت الذروة | مرتبطة بشروط X والخطة | مرتبطة بشروط Grok وخطته |
| الميزات متعددة الوسائط | حسب ما توفره X داخل المنتج | حسب ما يتيحه منتج Grok مباشرة |
| مزايا منصة X (توثيق/توزيع/إعلانات) | متاحة حسب خطة X | غير مشمولة تلقائيًا |

### 3.3 ما هو SuperGrok؟

SuperGrok في grok.com موجّه لمن يريد Grok كأداة أساسية يومية.  
أما مزايا X (المنصة) فهي مسار مختلف.

## 4) فرق جوهري يجب تذكره

- Premium+ يعني شراء تجربة X المتقدمة مع Grok داخل X
- SuperGrok يعني شراء تجربة Grok نفسها

تقارب الأسماء لا يعني تطابق العقد أو الفاتورة.

## 5) أخطاء شائعة

1. الاعتقاد أن SuperGrok يمنح Premium+ تلقائيًا  
   → غير صحيح، كل منتج له مسار اشتراك مستقل.

2. الاعتقاد أن أي خطة "تتضمن Grok" تكفي لكل الأحمال  
   → الوصول الأساسي شيء، وسعة الاستخدام الفعلية شيء آخر.

3. اعتبار تسمية "SuperGrok" داخل X مساوية دائمًا لاشتراك grok.com  
   → قد تتشابه الأسماء بينما تختلف الجهة والشروط.

4. التعامل مع الأسعار المنشورة كأنها ثابتة  
   → الأسعار والحدود قابلة للتحديث؛ صفحة الدفع هي المرجع.

5. افتراض أن API مشمول بالكامل داخل اشتراك المستخدم  
   → عادةً فوترة API للمطورين تُدار بشكل منفصل.

## 6) طريقة القرار بثلاثة أسئلة

1. أين تعمل أكثر: داخل X أم داخل Grok؟  
2. هل تحتاج مزايا X الخاصة (عرض/توزيع/تجربة منصة)؟  
3. هل استخدام Grok عندك خفيف أم إنتاجي ثقيل يوميًا؟

## 7) الخلاصة

- اشتراك X = شراء مزايا المنصة  
- اشتراك Grok = شراء مزايا المنتج  
- وقد تحتاج الاشتراكين معًا إذا كان استخدامك يجمع المسارين
